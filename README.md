# Sistema de detección de posibles fraudes en ofertas laborales mediante Web Scraping

## Descripción
Sistema desarrollado para recopilar ofertas de empleo mediante Web Scraping y analizar su contenido con técnicas de evaluación de riesgo y expresiones regulares, con el objetivo de identificar posibles anuncios de trabajo fraudulentos.
